## What Is EIP-1559?

**The Core Change:**

Before EIP-1559:

- Users bid gas fees in **auction**
- All fees go to **miners**
- Price volatility extreme
- **Overpaying common**

After EIP-1559:

- **Base fee** set algorithmically (burned ğŸ”¥)
- **Priority tip** to validators (small)
- **Predictable pricing**
- **ETH supply decreases** during high activity

**The Formula:**

```
Transaction Fee = Base Fee (burned) + Priority Tip (to validator)
```

**Base Fee Mechanics:**

- **Algorithmically adjusted** each block
- **Target**: 50% full blocks (15M gas)
- **If block >50% full**: Base fee increases
- **If block <50% full**: Base fee decreases
- **Maximum change**: 12.5% per block

**The Burn:**

Every transaction **destroys** ETH:

- High network activity = **more ETH burned**
- If burn > issuance = **net deflationary**
- **Permanent supply reduction**

## Why It Matters

**The Problem It Solved:**

**1. Fee Unpredictability:**

Old system (first-price auction):

- User bids 100 gwei
- Block has space for 50 gwei
- User **overpays by 2x**
- Happens constantly

New system:

- Base fee is 50 gwei
- User pays exactly 50 gwei (+ small tip)
- **No overpaying**

**2. Miner Manipulation:**

Old system:

- Miners could **reject transactions** to create scarcity
- **Artificial congestion** raises fees
- Gaming possible

New system:

- Base fee **algorithmically set**
- Can't be manipulated
- More fair

**3. Economic Model:**

Old system:

- ETH issuance **~4.5%** annually
- All fees to miners
- **Perpetual inflation**

New system:

- Fees **burned** (removed from supply)
- During high activity: **Burn > issuance**
- **Net deflationary** ğŸ”¥

## The Fee Market

**How It Works in Practice:**

**Low Activity (Normal):**

- Base fee: 15-30 gwei
- Priority tip: 1-2 gwei
- Total: ~20-35 gwei
- **Affordable** for most

**High Activity (NFT mint, market crash):**

- Base fee: 200-500 gwei
- Priority tip: 10-50 gwei
- Total: 250-550 gwei
- **Very expensive**

**Extreme Activity (rare):**

- Base fee: 1,000+ gwei
- Priority tip: 100+ gwei
- Total: 1,100+ gwei
- **$100+ per transaction**

**The Benefit:**

- **Predictable**: Know base fee before sending
- **Rational**: Algorithmic, not auction chaos
- **Fairer**: No systematic overpaying
